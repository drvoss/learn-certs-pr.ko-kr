# <a name="student-training-and-certification"></a>학생 교육 및 인증

## <a name="why-become-certified"></a>인증이 필요한 이유는 무엇인가요?

<div><br/>
기술은 업무의 미래를 바꾸고 있습니다. 직무와 업계에 관계없이 고용주는 인턴과 최근 졸업생이 오늘날의 하이브리드 또는 원격 작업 환경에서 일하는 데 필요한 디지털 역량을 갖춘 상태에서 입사하기를 기대합니다. 이러한 디지털 기술은 좋은 출발점이지만 진정 돋보이고 장래의 기회를 활용하려는 학생은 데이터, AI, 클라우드 기술의 능력이 필요합니다. Microsoft 인증을 얻기 위한 노력은 기술적 능력을 쌓는 데 그치지 않고 두각을 나타내고 더 많은 급여를 받는 데도 도움이 됩니다<sup>1</sup>. 나아가 Microsoft 인증을 통해 기술과 능력을 증명하고 새로운 기술을 배우려는 에너지와 헌신을 보여 줄 수 있습니다.</div>

## <a name="steps-to-getting-certified-and-your-dream-job"></a>인증과 꿈에 그리는 직업을 얻는 단계

![프로세스 - 6단계](images/step-process.png)

### <a name="1--sign-in-to-learn-and-get-learning"></a>1. Learn에 로그인하고 학습하세요.

[Microsoft Learn](https://aka.ms/learn)에 로그인하고 학생 프로필을 만듭니다. 자신의 속도와 일정에 맞춰 새로운 개념을 마스터합니다. Learn에 로그인하면 학습 활동 진행 상황 추적, 모듈 컬렉션 만들기 및 공유, 점수 및 성취 계산, 무료 Azure 리소스 사용, 맞춤형 권장 사항 가져오기 등의 독점적 기능에 액세스할 수 있습니다. 

### <a name="2--verify-your-academic-profile-to-get-access-to-student-discounts"></a>2. 학생 할인을 이용하려면 교육 기관 프로필을 확인합니다.

Microsoft는 대부분의 국가/지역에서 Microsoft 인증 시험 학생 할인을 제공합니다. 학생 할인을 받으려면 먼저 Microsoft 인증 프로필에서 자신을 학생으로 식별한 다음 학업 상태를 확인해야 합니다. 인증이 완료되면 시험 등록 시 학생 할인을 받을 수 있습니다. 자격에 대해 [자세히 알아보고](/learn/certifications/student-discounts) Microsoft 인증 프로필에서 학생임을 입증하는 방법에 대한 단계별 지침을 찾아보세요.

### <a name="3--determine-which-microsoft-certification-is-right-for-you"></a>3. 적합한 Microsoft 인증을 결정합니다.

Microsoft 인증은 직무와 역할에 대한 업계 분석을 기반으로 하며, 최신 기술에 따라 지속적으로 업데이트됩니다. Microsoft 인증은 신뢰도를 향상시키며, 전 세계에서 인정되고 업계에서 보증하는 기술 숙달의 증거를 제공하므로 전문적 이점을 얻을 수 있습니다. Microsoft 인증 획득은 비즈니스 가치 및 자율성과 직결되는 재능, 기술, 경험을 고용주와 동료에게 인정받을 수 있는 간단한 방법을 제공합니다.

Microsoft 인증은 먼저 기본적인 기술을 구축한 다음 업계 역할을 수행하는 데 필요한 보다 깊이 있는 기술적 능력을 쌓는 방법을 제공합니다. 또한 Microsoft 인증은 Azure부터 AI, 데이터 분석, 사이버 보안에 이르는 기술 주제를 포괄합니다. 관심이 있는 경력에 맞는 인증을 확인하는 방법에 대해 [자세히 알아보세요](https://aka.ms/TrainCertPoster). 

#### <a name="fundamentals-certifications"></a>Fundamentals 인증

Fundamentals 인증은 Microsoft 클라우드 및 비즈니스 애플리케이션 서비스에 대한 기본 수준 지식을 다루며, 기술 분야 경력을 시작하거나 고려하고 있는 학생들에게 적합합니다.
- [Azure Fundamentals(AZ-900)](/learn/certifications/exams/az-900)
- [Azure AI Fundamentals(AI-900)](/learn/certifications/exams/ai-900)
- [Security Compliance and  Identity Fundamentals(SC-900)](/learn/certifications/exams/sc-900)

[모든 Fundamentals 인증을 보려면 클릭하세요.](/learn/certifications/browse/?type=fundamentals)

#### <a name="role-based-certifications"></a>역할 기반 인증

역할 기반 인증은 Microsoft 클라우드 및 비즈니스 애플리케이션 서비스에 대한 Associate 수준 지식을 포괄합니다. 중요한 업무 기술 학습을 시작하려는 학생에게 적합합니다.
- [Microsoft Azure 관리자(AZ-104)](/learn/certifications/exams/az-104)
- [Designing and Implementing Microsoft Azure AI solution(AI 102)](/learn/certifications/exams/ai-102)
- [Microsoft Azure 보안 기술(AZ-500)](/learn/certifications/exams/az-500)

[모든 역할 기반 인증을 보려면 클릭하세요.](/learn/certifications/browse/?branch=main&type=role-based)

### <a name="4-learn-prepare-get-certified"></a>4. Learn, 준비, 인증 받기

#### <a name="i-learn"></a>i) Learn

a. [Microsoft Learn](https://aka.ms/learn)은 언제든지 온라인으로 액세스할 수 있는 무료 실습 브라우저 기반 학습 경로를 제공합니다. 재미있는 대화형 모듈, 샌드박스, 학습 경로를 통해 실용적인 기술을 개발하려면 여기서 시작하세요. 모든 Microsoft Fundamentals 및 역할 기반 인증에 Microsoft Learn 학습 경로를 사용할 수 있습니다.  
b. Exam Cram 녹화  
  이 주문형 녹화는 시험을 구성하는 인증 목표를 빠르게 검토하고 시험 응시를 준비하는 데 필요한 주요 영역을 설명합니다.  

  1. [Azure Fundamentals(AZ-900) Exam Cram](https://aka.ms/ExamCramAZ900)  
  2. [Azure AI Fundamentals(AI-900) Exam Cram](https://aka.ms/ExamCramAI900)  
  3. [Azure Data Fundamentals(DP-900) Exam Cram](https://aka.ms/ExamCramDP900)  
  4. [Power Platform Fundamentals(PL-900) Exam Cram](https://aka.ms/ExamCramPL900)  
  5. [Security, Compliance and Identity Fundamentals(SC-900) Exam Cram](https://aka.ms/ExamCramSC900)  

다. [MVTD(Microsoft Virtual Training Days)](https://www.microsoft.com/en-ie/training-days/) – 프레젠테이션, 데모, 토론, 실습 워크샵의 조합을 제공하는 Microsoft 전문가와 함께하는 1일 및 2일 가상 교육 이벤트를 활용하세요.  

#### <a name="ii-prepare"></a>ii) 준비

Microsoft Official Practice Test – Microsoft 인증 시험 합격률을 크게 높이려면 Microsoft Official Practice Test에 응시하세요. 시험 목표에 대한 지식을 쌓고 정답과 오답에 대한 자세한 설명을 확인하세요. MeasureUp과 함께하는 Microsoft Official Practice Test에 대해 [자세히 알아보세요.](https://www.measureup.com/?msclkid=0163a21ad4b818faac51c3b49589eaea&utm_source=bing&utm_medium=cpc&utm_campaign=0.%20Brand%20US&utm_term=measureup&utm_content=Brand)

#### <a name="iii-get-certified"></a>iii) 인증 받기

a) 학생 가격 할인 - 학생 할인을 받으려면 등록 중에 학생임을 확인해야 합니다. Microsoft Learn의 시험 페이지에서 직접 시험을 예약합니다. [학생 할인에 대해 자세히 알아보려면 이 링크를 방문하세요.](/learn/certifications/student-discounts)  
b) 교수 또는 경력 서비스 카운슬러에게 문의하여 학교에서 학생에게 무료 또는 할인된 인증 시험 기회를 제공하는지 확인합니다.  
c) [MVTD(Microsoft Virtual Training Day)](https://www.microsoft.com/en-ie/training-days/)에 참석하여 무료 인증 시험 쿠폰을 받을 수 있습니다. MVTD 사이트에서 이 기회에 대해 자세히 알아보세요.  

### <a name="5--get-college-credit-for-your-certification-exam"></a>5. 인증 시험에 대한 대학 학점 받기

좋은 소식 Microsoft 시험을 통과하고 Microsoft 인증을 획득한 학생은 대학 학점을 받을 수 있습니다. Microsoft 인증을 받아 얻은 대학 학점을 사용하여 학비를 절약하고 학위를 더 빠르게 마친 Sam Jones와 Jason Powell에 대해 [읽어 보세요](https://techcommunity.microsoft.com/t5/microsoft-learn-blog/students-get-creative-over-break-earn-microsoft-certifications/ba-p/1820647). [이와 똑같이 할 수 있도록 이들이 사용한 도구와 리소스를 검토하세요.](/learn/certifications/college-credit?wt.mc_id=msle_learn_blog_wwl)  

### <a name="6-post-your-certification-on-linkedin"></a>6. LinkedIn에 인증을 게시하세요.
새 자격 증명을 축하합니다.

인증을 받으면 LinkedIn 프로필에서 자신의 기술을 업데이트하세요.

LinkedIn에 새 인증을 게시해야 하는 이유는 무엇인가요?

전문적 성취를 보여 주면 잠재적 고용주의 눈에 띄는 데 도움이 될 수 있습니다.  인증은 LinkedIn 프로필에서 지식과 전문적 성취를 보여 주는 좋은 방법입니다.

LinkedIn 프로필에서 기술과 인증을 보여 주는 방법에 대해 [자세히 알아보세요](https://www.linkedin.com/help/linkedin/answer/44644/manage-licenses-certifications?lang=en).
