# <a name="start-with-our-new-workload-certifications-to-become-a-microsoft-365-enterprise-administrator"></a>새로운 워크로드 인증을 시작하여 Microsoft 365 Enterprise 관리자 되기

Liberty Munson에서 03 Dec 2018 게시일 **(Microsoft)**

___

2019 년 1 월 17 일, 2019 년 4 월 11 일: 아래 빨강의 업데이트를 참조 하세요.

Microsoft 365 워크 로드 인증을 통해 [Enterprise 관리자 전문가에 게 Microsoft 365 인증](https://www.microsoft.com/learning/m365-enterprise-administrator.aspx)을 받을 수 있습니다. 이 블로그를 팔로 하는 경우, [Enterprise 관리자](https://www.microsoft.com/en-us/learning/community-blog-post.aspx?BlogId=8&Id=375171) 및 [최신 바탕 화면](https://www.microsoft.com/en-us/learning/community-blog-post.aspx?BlogId=8&Id=375176)에 대 한 Microsoft 365 인증 된 베타 시험을 최근에 실행 한 것입니다. 더 많은 정보를 제공 하는 것을 알고 있으며,이를 위해 몇 가지 이야기를 막대인 경우도. 여기서는 다음 부분을 이야기 합니다. . .

[최근 Deloitte 설문 조사](https://www2.deloitte.com/us/en/pages/deloitte-private/articles/technology-trends-middle-market-companies-survey.html) 는 고용주에 대 한 주요 우선 순위로 강조 표시 됩니다. 더 작은 방법으로 더 많은 작업을 수행 하는 것은 아니지만 전반적으로 더 많은 경쟁 우위를 사용 하는 것이 중요 합니다. 실제로 고용주의 46%는 디지털 중단의 결과로 채용을 확장 하도록 설계 되었습니다. Microsoft cvp Brad Anderson가 Ignite에 강조 표시 되 면 회사는 [Microsoft 365로 수준을](https://www.microsoft.com/en-us/microsoft-365/blog/2018/09/28/why-and-how-you-simplify-it-with-microsoft-365/)높이고 공동 작업을 통해 민첩성을 높이고 혁신을 진행 합니다. 이 질문은 Microsoft 365에서 인증 해야 하는 것은 아니지만 워크 로드 인증 중 어떤 것이 적절 한지에 대 한 것입니다.

인증 과정을 시작 해 보겠습니다.

___

***Microsoft 365 인증: 기본* ([시험 900](https://www.microsoft.com/learning/exam-MS-900.aspx); 지금 사용 가능)**

최신 작업 공간에 대 한 작업을 시작 하 고 기술에서 강력한 배경 지식이 없는 경우이를 시작 하는 것이 좋습니다. 이 시험은 고려 사항, 일반적으로 클라우드 서비스를 채택 하는 이점, SaaS (Software as a Service) 클라우드 모델, Microsoft 365 클라우드 서비스 제공을 구현 하 여 얻은 옵션과 이점에 대 한 정보를 확인 합니다. Microsoft 365의 기본 사항을 확실 하 게 이해 하 고 있으며, [최신 데스크톱 관리자,](https://www.microsoft.com/learning/modern-desktop.aspx) Teamwork 관리자, 메시징 관리자 및 궁극적으로 [Microsoft 365 Enterprise 관리자](https://www.microsoft.com/learning/m365-enterprise-administrator.aspx)와 같은 Microsoft 365 작업 역할을 준비 하는 것이 확인 됩니다. 이 시험은 Office 365, Microsoft Intune, Azure Information Protection (aip) 및 Windows 10과 같은 클라우드 컴퓨팅 및 기술 시험에 대 한 **선택적** 기반이.

___

***[Microsoft 365 인증: 최신 데스크톱 관리자 연결](https://www.microsoft.com/learning/modern-desktop.aspx)* (시험 [md-100](https://www.microsoft.com/learning/exam-MD-100.aspx) 및 [MD-101](https://www.microsoft.com/learning/exam-MD-101.aspx); 지금 사용 가능)**

최신 데스크톱 관리자는 인증을 연결 하 여 최신 데스크톱 솔루션을 구현 하는 첫 번째 단계인 Windows 10를 배포 하기 위한 요구 사항 및 메커니즘을 신속 하 게 제공할 수 있습니다. 또한 사용자 환경을 개선 하 고 생산성을 높일 수 있도록 전체 업그레이드, Windows Autopilot 및 통합 id 솔루션을 사용 하 여 배포를 간소화 하는 방법을 알아봅니다.

2019 년 3 월 31 일에 만료 되기 전에 [시험 70-698](https://www.microsoft.com/learning/exam-70-698.aspx)  를 통과 한 경우 최신 데스크톱 인증을 얻기 위해 2019 년 7 월 31 일 이전에 시험 (MD 101)을 한 번만 수행 하면 됩니다. 2019 년 8 월 1 일부 터, md-101 및 md 101 모두 필수 시험 이며 Microsoft 365 인증을 받기 위한 유일한 경로입니다 [. 최신 데스크톱 관리자](https://www.microsoft.com/learning/modern-desktop.aspx) 는 인증을 연결 합니다.

___

***[Microsoft 365 인증 됨: 메시징 관리자 연결](https://www.microsoft.com/learning/m365-messaging-administrator.aspx)* (시험 [MS-200](https://www.microsoft.com/learning/exam-MS-200.aspx), [201](https://www.microsoft.com/learning/exam-MS-201.aspx), ms [-202](https://www.microsoft.com/learning/exam-MS-202.aspx), 지금 사용 가능)**

Exchange 관리자는 사용자를 위한 인증입니다. 기술에는 조직 내부와 외부 모두의 통신을 위한 다양 한 옵션을 제공 하지만 확장 된 액세스에는 위험이 있습니다. 회사에는 이러한 위험을 완화 하는 방법을 알고 있는 관리자가 필요 합니다. 이 인증은 Microsoft 365 업계 최고의 도구 제품군을 사용 하 여 규정 준수 및 보안 통신 솔루션을 설계 하 고 관리 하는 데 필요한 기술이 있는지 확인 합니다.

[시험 70-345: Microsoft Exchange Server 2016 디자인 및 배포](https://www.microsoft.com/learning/exam-70-345.aspx)를 통과 한 경우 시험 MS-202: Microsoft 365 메시징 관리자 인증 전환 시험을 통해이 새로운 역할 기반 인증을 한 번에 수행할 수 있습니다.

___

***[Microsoft 365 인증 됨: Teamwork 관리자 연결](https://www.microsoft.com/learning/m365-teamwork-administrator.aspx)* (시험 [MS-300](https://www.microsoft.com/learning/exam-MS-300.aspx), [301](https://www.microsoft.com/learning/exam-MS-301.aspx), ms [-302](https://www.microsoft.com/learning/exam-MS-302.aspx), 지금 사용 가능)**

공동 작업 도구는 최신 작업 공간에서 핵심은 됩니다. 이전 보다 훨씬 더 빠르게 데이터에 액세스 하 고 수정할 수 있어야 하 고, 안전 하 고 확장 가능 하며 분산 된 인력을 사용 하지 않는 통합 솔루션이 필요한 Teams. 이 인증은 회사의 가장 까다로운 생산성 퍼즐 해결 하는 세련 되 고 안전 하며 규정을 준수 하는 솔루션을 설계 하는 기술의 유효성을 검사 합니다.

[시험 70-339: Microsoft SharePoint Server 2016 관리](https://www.microsoft.com/learning/exam-70-339.aspx)를 통과 한 경우 [MS-302](https://www.microsoft.com/learning/exam-MS-302.aspx): Microsoft 365 Teamwork 관리자 인증 전환 시험을 통해 단일 시험 만으로이 새 인증을 받을 수 있습니다.

___

***[인증 Microsoft 365: 보안 관리자](https://www.microsoft.com/learning/m365-security-administrator.aspx)는 * ([시험 500](https://www.microsoft.com/learning/exam-MS-500.aspx); 지금 사용 가능)를 연결 합니다.**

정규화 된 보안 관리자의 감독은 IT 구현 및 작업의 성공에 매우 중요 하지 않으며, [Forbes는 기술에서 가장 많은 주문형 역할 중 하나](https://www.forbes.com/sites/forbestechcouncil/2018/08/09/the-cybersecurity-talent-gap-is-an-industry-crisis/) 이며 인재 격차는 계속 증가 하 고 있음을 보고 합니다. 이 인증은 더 복잡 한 사이버 보안 위협 으로부터 회사를 보호 하는 데 필요한 기술을 보유 하 고 있으며 [점점 계층화 되 고 다양 한 데이터 규정 준수를 유지](https://www.microsoft.com/en-us/microsoft-365/blog/2018/09/25/start-using-microsoft-365-to-accelerate-modern-compliance/)하는 데 필요한 기술을 사용자에 게 보여 줍니다.


___

Microsoft의 인증 경로를 선택하면 현재 고용주와 잠재적 고용주 모두에게 두각을 나타내는 데 도움이 되며 민첩성과 경쟁력을 유지하는 데 필요한 IT 기술을 보유하고 있음을 입증할 수 있습니다. 또한 Microsoft에서 이를 증명할 수 있도록 도와드립니다. 새로 디자인된 [배지](https://www.microsoft.com/learning/badges.aspx)를 한 번만 클릭하면 동료와 고용주가 귀하의 전문 지식을 확인하고 귀하의 자격에 대해 자세히 알아볼 수 있습니다. 이를 통해 동료와 고용주에게 두각을 나타내고 입증된 기술과 지식을 기반으로 [새로운 고용 기회를 얻을 수 있습니다](https://www.microsoft.com/en-us/learning/community-blog-post.aspx?BlogId=8&Id=375167).

## <a name="how-do-i-get-started"></a>시작하려면 어떻게 해야 하나요?

Microsoft의 [기술 인증](https://www.microsoft.com/learning/browse-new-certification.aspx)을 방문하여 사용 가능한 인증에 대해 자세히 알아보세요. 어떤 인증에 응시할지 결정했으면 Microsoft Learning 파트너 중 하나를 통해 제공되는 [강사 진행 교육](https://www.microsoft.com/learning/partners.aspx)과 [Microsoft Learn](https://docs.microsoft.com/learn/)의 무료 온라인 옵션을 포함하여 시험 준비를 도와주는 적합한 교육 옵션을 찾아보세요.

**질문이 있으세요?** 다시 디자인된 프로그램에 대한 개요는 [역할 기반 인증 블로그 공지](https://www.microsoft.com/en-us/learning/community-blog-post.aspx?BlogId=8&Id=375161)를 확인하세요.  또한 [시험 FAQ](https://www.microsoft.com/learning/certification-exam-policies.aspx)에서 응시할 준비가 되었을 때 필요한 내용에 대한 자세한 내용을 확인할 수도 있습니다.

그리고 Microsoft 365 인증 Enterprise 관리자 전문가 인증을 획득 하는 방법에 대 한 나머지 이야기를 계속 조정 하세요.
